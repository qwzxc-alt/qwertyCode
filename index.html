Помощь по qwertyCode

Основные возможности

qwertyCode - это мощный веб-интерпретатор для HTML, CSS и JavaScript, который работает прямо в вашем браузере.

Быстрые клавиши
  • Ctrl+Enter - Выполнить код
  • Ctrl+S - Сохранить проект
  • Ctrl+Shift+S - Скачать проект
  • Ctrl+1,2,3 - Переключение между редакторами
  • Ctrl+/ - Закомментировать/раскомментировать
Советы по использованию
  • Используйте кнопку "Развернуть" для фокусировки на одном редакторе
  • Дважды щелкните на разделителе, чтобы сбросить размер панелей
  • Консоль отображает ошибки и предупреждения вашего кода
  • Вы можете делать скриншоты результата
